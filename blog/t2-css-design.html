    Margin, Borders and Padding

    What is the difference between Margin, Border, and Padding?

    19 July 2015

    Margin

    When applying a Margin to an element, it creates white space around the outside the border of the element. The white space is transparent so elements can still interact with it. When adding a margin to a page there are many ways to do it in the css. margins can be added to every individual side of the element or just one, and the margins for all sides can be different. It is also important to note that margins auto collapse, meaning that if there were 2 elements with 1em margin on each, the margin would be 1em and not 2em.

    Ways to write margin:

    margin: 1px 1px 1px 1px; : this applies a 1px margin to each side of the element

    margin-top:1px; margin-bottom:1px; margin-left:1px; margin-right:1px;

    This also applies a 1px margin to each side, however it become easier to read.

    Borders

    Borders are located between the margin and the padding and can be any size and color the developer wishes. Borders also have many different styles that can be applied to them. These styles include Dotted, dashed, solid, double solid and others.

    The way you would write a border is a little different to margin and padding:

    border: 1px 1px 1px 1px solid black;

    This creates a 1px solid black border around the element that it is applied to.

  • border-top: 1px dotted blue;
  • border-bottom: 1px dashed red;
  • border-left:1px double purple;
  • border-right:1px solid pink;

    • This would create a border with 1px on each side however each side would be both a different style and color.

    Padding

    Unlike margin, a padding is applied to the inside of the border. and it clears an area around the content of the element. Also unlike margin, it adds, so if there were two elements with 1em padding on each, The padding would be 2em instead of 1em. The thing that both Margin and Padding have in common however is that they are both transparent.

    Ways to write margin:

    padding: 1px 1px 1px 1px;

    This applies a 1px padding to each side of the element

  • padding-top:1px;
  • padding-bottom:1px;
  • padding-left:1px;
  • padding-right:1px;

    • This also applies a 1px padding to each side, however it become easier to read.
